### 💻  프로젝트 설명
개인 맞춤형 식단을 추천해주는 식단 관리 어플리케이션

* 사진과 텍스트로 식사 기록 및 조회
* 사용자의 선호와 이전 기록을 통한 개인 맞춤형 식단을 추천
* 영양소 비율에 맞춰 식사하고 있는지 통계 제공
* 몸무게 추적 및 예상 그래프 제공
